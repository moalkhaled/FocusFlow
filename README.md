# FocusFlow

FocusFlow is a web application for task management developed using PHP and MySQL.

The application allows users to:
- Create tasks
- Update tasks
- Delete tasks
- Manage task categories
- Track task status

## Technologies

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Installation

1. Install XAMPP.
2. Copy the project folder to htdocs.
3. Start Apache and MySQL.
4. Import the database from db/schema.sql.
5. Open:

http://localhost/FocusFlow
